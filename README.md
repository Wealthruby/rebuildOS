RebuildOS
RebuildOS is a custom WordPress plugin for Rebuild With Intention.
It provides a free, private, self-directed rebuilding dashboard using the shortcode:
[rebuild_os]
The tool helps users:
•	complete daily rebuild check-ins
•	track urges and high-risk moments
•	identify trigger patterns
•	run an emergency reset protocol
•	complete relapse autopsies without shame
•	review weekly progress
•	build replacement identity systems
•	export their data
Product Positioning
RebuildOS is not just a habit tracker.
The goal is not only to count clean days.
The goal is to understand the system that keeps breaking and rebuild it with intention.
Core Principles
1.	Private by default.
2.	Free at the core.
3.	No shame language.
4.	No cure claims.
5.	Data should help users understand patterns.
6.	Recovery support should be practical and grounded.
7.	Guest users should be able to use the tool immediately.
8.	Logged-in users should be able to save progress across devices.
